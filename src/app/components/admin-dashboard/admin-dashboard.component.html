ℹ️ Como gerenciar artigos neste Blog Estático

Este blog funciona de forma estática (Jamstack) carregando arquivos Markdown locais. Para gerenciar os posts:

  • Criar: Clique em "Novo Artigo", escreva o post, veja o preview e copie os blocos gerados para o diretório public/articles/.
  • Editar: Clique no ícone de lápis (Editar) de um artigo para recarregar as informações no editor, faça os ajustes e copie o código atualizado.
  • Excluir: Clique em lixeira (Excluir) para ver quais referências remover no arquivo articles.json e deletar no disco.
